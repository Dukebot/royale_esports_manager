# royale_esports_manager

Aplicación web para gestionar un club de e-sports de Clash Royale.

Entorno de desarrollo: Eclipse

Tipo de proyecto: Dynamic web project

Librerías:
* Java Persistence Api
* Postgre sql driver
* Eclipse Link
* JTL

Módulos actuales:
* Cartas
* Mazos
* Jugadores
* Equipos

Módulos pendientes:
* Campeonatos
* Partidos
